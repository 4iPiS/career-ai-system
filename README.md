# Career AI - Умная система профориентации

Искусственный интеллект для рекомендации направлений обучения на основе анализа профиля студента.


## Быстрый старт

### 1. Установите зависимости
pip install -r requirements.txt


### 2. Обучите модель
python train_model.py


### 3. Получите рекомендации
python career_ai.py examples/student_ivan.csv
